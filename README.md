## 📌 Overview
A robust Java-based application designed to streamline student academic record management. The system enables efficient input, processing, and display of student data while automatically calculating grades and generating structured report cards using object-oriented principles.

---

## ✨ Key Features
✔️ Add and manage student records  
✔️ Automated grade calculation system  
✔️ Structured report card generation  
✔️ Clean and modular OOP-based design  
✔️ Simple and intuitive command-line interface
